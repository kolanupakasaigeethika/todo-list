# 📝 Todo Dashboard App

A full-stack Todo Dashboard application built using Node.js, Express, MongoDB, and Vanilla JavaScript. This project helps users manage daily tasks efficiently with a clean and interactive dashboard interface.

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 📅 Add due dates to tasks
* 📊 Filter tasks (All / Pending / Completed / Overdue)
* 🌙 Dark mode with theme persistence
* ⚡ Real-time updates without page reload

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript (Vanilla JS)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (MongoDB Compass)
* **API Testing:** Postman

## 🔗 API Endpoints

* `GET /api/todos` → Fetch all tasks
* `POST /api/todos` → Create a new task
* `PUT /api/todos/:id` → Update task (edit/complete)
* `DELETE /api/todos/:id` → Delete task

## ⚙️ How to Run the Project

### 1️⃣ Run Backend

```
cd backend
npm install
npm start
```

### 2️⃣ Run Frontend

```
cd frontend
npx serve .
```

👉 Open in browser:
http://localhost:3000
